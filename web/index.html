Hồ sơ của nhà thầu

Có thể chọn một file để đối chiếu riêng với phụ lục, hoặc nhiều file để so sánh thêm về giá.

Bắt buộc
Chưa có hồ sơ nhà thầu nào được chọn.
Chọn 1 file: kiểm tra hạng mục, mã, đơn vị, khối lượng và yêu cầu trong phụ lục. Chọn từ 2 file: hệ thống bổ sung so sánh giá giữa các nhà thầu.
