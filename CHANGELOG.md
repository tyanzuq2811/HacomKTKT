# Changelog v7.2.0

- Chuyển chế độ HSDT–HSDT sang so sánh ngang hàng thật sự.
- Không dùng file đầu tiên làm baseline.
- Ghép từng cặp theo cả hai chiều rồi hợp nhất bằng đồ thị one-to-one.
- Hỗ trợ 2, 3 hoặc nhiều nhà thầu.
- Công thức chênh lệch dùng toàn bộ giá trị, không thiên vị một nhà thầu.
- So sánh giá, khối lượng, thành tiền, thành phần giá, tên và thông số kỹ thuật.
- Báo cáo ghi rõ giá trị từng nhà thầu, phần trăm và lý do.
- Viết lại giao diện theo ba bước, dùng ngôn ngữ dễ hiểu.
- Loại bỏ API HSMT-vs-HSDT khỏi giao diện Phase hiện tại.
- Không dùng Docker hoặc Docker Compose.
